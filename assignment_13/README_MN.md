# Долоо хоног 13: Гүйцэтгэлийн туршилт ба Аюулгүй байдлын үндэс

Энэхүү хавтас нь 13-р долоо хоногийн бие даалтын ажлын хүрээнд хийгдсэн скриптүүд болон тайлбарыг агуулна.

## 1. Гүйцэтгэлийн туршилт (`performance_test.py`)

Энэхүү Python скрипт нь вэбсайтын гүйцэтгэлийг шалгах зорилготой. Олон хэрэглэгч зэрэг хандалт хийж байгааг дуурайлган (concurrent users), хариу өгөх хугацааг хэмждэг.

### Ажиллуулах заавар:
1.  Python суулгасан байх шаардлагатай.
2.  Шаардлагатай санг суулгах:
    ```bash
    pip install requests
    ```
3.  Скриптийг ажиллуулах:
    ```bash
    python performance_test.py
    ```
4.  **Тайлбар**: Скрипт доторх `target_url` хувьсагчийг өөрчилж өөрийн deploy хийсэн вэбсайт (жишээ нь: `https://mongolia-travel-planner.onrender.com`) эсвэл `localhost`-оо шалгаж болно.

## 2. Аюулгүй байдлын жишээнүүд (`security_examples.py`)

Энэхүү скрипт нь вэб хөгжүүлэлт дэх нийтлэг аюулгүй байдлын алдаанууд болон тэдгээрийг хэрхэн зөв шийдвэрлэх жишээнүүдийг харуулна.

### Агуулга:
*   **SQL Injection**:
    *   *Муу жишээ*: Хэрэглэгчийн оролтыг шууд SQL query-д залгах.
    *   *Сайн жишээ*: **Parameterized Query** ашиглан өгөгдлийг тусгаарлах.
*   **XSS (Cross-Site Scripting)**:
    *   *Муу жишээ*: Хэрэглэгчийн оруулсан текстийг шууд HTML болгон харуулах.
    *   *Сайн жишээ*: **Input Sanitization** (HTML escaping) хийж тусгай тэмдэгтүүдийг хувиргах.
*   **Нууц үг хадгалах**:
    *   *Plain text*: Нууц үгийг илээр хадгалах (Хэзээ ч хийж болохгүй!).
    *   *Hashing*: SHA256 гэх мэт алгоритм ашиглах.
    *   *Salted Hashing*: Hash дээр санамсаргүй утга (salt) нэмж хамгаалалтыг сайжруулах.
    *   *Bcrypt*: Нууц үг хадгалахад зориулагдсан орчин үеийн, найдвартай алгоритм.

### Ажиллуулах заавар:
1.  `bcrypt` санг суулгах:
    ```bash
    pip install bcrypt
    ```
2.  Скриптийг ажиллуулах:
    ```bash
    python security_examples.py
    ```

## Дүгнэлт

Энэхүү даалгавраар бид системийн тогтвортой байдлыг хэрхэн шалгах (Performance Testing) болон түгээмэл аюулгүй байдлын эрсдэлүүдээс (SQLi, XSS) хэрхэн сэргийлэх, хэрэглэгчийн нууц үгийг хэрхэн зөв хадгалах талаар практик мэдлэг олж авлаа.
